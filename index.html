إعداد Firebase

الخطوة الأولى — اربط البرنامج بقاعدة البيانات

📋 خطوات إنشاء مشروع Firebase:
1. اذهب إلى Firebase Console
2. Add Project → اختر اسم → Continue
3. من القائمة: Build → Authentication → Get Started → فعّل Email/Password
4. من القائمة: Build → Firestore Database → Create database → اختر Production
5. Project Settings (⚙️) → نزّل تحت → Add Web App (</>) → سجّل التطبيق
6. انسخ القيم من firebaseConfig والصقها هنا 👇
